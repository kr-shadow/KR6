* 御製逍遙詠序　　　　　輕

「玄元大道理包深遠。」
【(玄元者虛極之妙本天地之元精淳
湛一真凝然本寂所以伯陽宗其至道教演五千囊括虛無可謂深遠矣)。】
「
逍遙
至論義貫精微。」
【(逍遙者懸解之名冲虛曠達之義大鵬翔乎寥
廓至人遊於無為放意於自得之場別物物任其性乃可妙契三無化周六合使澆
漓之俗復淳朴之風精微之𣅀其在於斯)。】
「
朕聽覽之餘留心
緗帙。」
【(万機之餘不倦披覽雖漢武乙夜觀書無以過也)。】
「
三墳五典
頗識指歸。」
【(三墳者三皇之書二皇之道大於五帝五典者五帝之書
五帝之道可以常行是以聖人皆達指歸妙盡其要者也)。】
「
金匱玉凾亦
探玄奧。」
【(金匱玉凾者太公黃石公所菩秘法也皆論六韜三略富國安
民養生鍊神機權要妙雖其淵契盡皆探賾矣)。】
「
嘗念寡昧於道
不達罔測津涯故發此詠題庶幾明
白坦然矣。」
【(夫欲明至道之極故著逍遙歌詠以暢宸襟而猶道成不
宰謙卑益光遂發以罔測之辤尚假以庶幾之語故老子云江海能為百谷王宥以
其善下故也)。】
「
夫道之妙也五行能之運化。」
【(道本
冲寂應用無方妙在卷舒成乎變化故在天曰五星在物曰五行經緯彝倫招蘓
彚也)。】
「
三才可以變通。」
【(三才著象皆從其道道變無窮乃道乎天
地人也)。】
「
虎嘯龍吟陽唱陰和。」
【(此明至道自然之理且龍
者水畜雲者水氣虎者陰物風者陰精以類相召自相符契故易云同聲相應同氣
相求此其義也)。】
「
窮其秘也神明可辯於情狀。」
【(道之
秘也神全形具體與物𡨋原始要終窮理盡性推其情狀何深奧之不達哉)。】
「
究
其精也剛柔自符於易簡。」
【(廓然無間妙極太虛
晝夜以之晦明寒暑因而來往所以𠃵剛坤柔自合易簡俾不為而善始不勞而善
成苟非道之至微孰能詳放此乎)。】
「
而惑者昧於修已攻乎
異端俟駕鶴而𩥵鸞猶繫風而捕影。」

【(蒙昧之人不達至道謂神仙之可學顧囂塵而欲棄或置盤承露或築臺通天求驂
駕鸞鶴亦猶繫風捕影不可得而及也)。】
「
老氏五千之旨尚不
述於神仙。」
【(老子所著五千言且不述神仙之事但欲顯道之玄極歸
乎真源者矣)。】
「
莊周九萬之談理更超於言象。」

【(南華云北溟有鵾化而為鵬九萬一程六月一息盖舉其至大以類至散皆齊物理
也是知事契精微理超言象誠哉莊生所諭實至道之樞要逍遙之妙旨也)。】
「
先
賢穎脫皆知道而不言。」
【(自昔達道之士既知道在
於我所以不言老子云知者不言斯不誣矣)。】
「
庸愚困蒙豈希
顏而議學。」
【(庸昧之者妄學神仙有類希顏欲齊亞聖不亦遠乎)。】
「
昔
秦始皇邪見所惑徒望蓬萊。」
【(始皇帝為邪所
惑道德不修妄學神仙欲求度世遂遣徐福往彼蓬萊採其靈藥徐福以無驗而不
返始皇猶未悟而懸望矣)。】
「
漢武帝民力是輕莫尋員
嶠。」
【(漢武神智獨穎英武無倫尤好方士禱祈山岳後感王母降於帝宮授以六甲
辰生之訣自謂必登仙品芥視黎甿崇峻臺館坈戮降服既失上仙之誡莫尋貪嶓
之靈矣)。】
「
斯皆不修仁德求利好貪。」
【(守位曰仁好生
曰德既殫民力復慕仙遊比轅適越良可悲矣)。】
「
事蓋不經良足
太息。」
【(事不師古動越典經百世之後斯過不泯誠堪太息者哉)。】
「
今之
所述其殆庶乎。」
【(殆近也庶幾道也今茲所製或象陰陽則𠃵坤
而演妙或明金水法离坎以談玄追莊老清淨之遐蹤継藏軒淳元之絕䡄近乎至
道樂矣大方且非慕彼神仙之術也)。】
「
或近取諸身。」
【(或內養純和之
氣靜修真一之心動合靈源咸歸妙本也)。】
「
或遠取諸物。」
【(或以日精
月華鈆水金液託其象而成其言也)。】
「
其詣理也不可以一途
執。」
【(窮其理也則必有所詣詣其理者乃能成天下之務也至若適時知變在乎幾
也又焉可以一途而執哉)。】
「
其筌意也不可以一理求。」

【(徵其意也則必因於筌筌其意者乃能通天下之志也至若窮神極化在乎深也又
焉可以一理而求哉)。】
「
探𧷤幽微。」
【(探𧷤之妙通幽洞微)。】
「
㕘詳法
度。」
【(極思㕘詳規矩法度)。】
「
智者見之謂之智仁者
見之謂之仁。」
【(俾夫智者覽逍遙之旨則動而無窮也仁者得逍遙之道
則澹乎無為也)。】
「
玄覽而可以照三清。」
【(上清玉清太清謂之
三清也玄覽心照也混尓真心朗然玄照越彼三天之上也)。】
「
旁行而可
以通萬有。」
【(不疾而速未兆而萌仁智之所莫拘動靜之所一致既周
行無礙又何萬有而不通哉)。】
「
夫詩頌歌辭華而不實。」

【(汎而論之不務其實則或傷於華也)。】
「
上不足以補時政之
闕失。」
【(時政之本務求其實必尚華言終無所補也)。】
「
次不足以
救蒼生之弊病。」
【(欲彼黎元咸躋富壽者義存道本也必尚
彼浮詞誠不足以救蒼生之弊病矣)。】
「
假有獨擅其名弗精
於用。」
【(辭也者所以通萬物之情也假有專其述作之名昧彼述作之用所
謂華而不實也)。】
「
義同畫餅棄不足珍。」
【(文彩燦然形容
儼若但堪恱目何可充膓就不足珍也)。】
「
爭如立至當之言
以濟於時世者哉。」
【(夫聖人之意其不可得而見也故觀之於
象而形之於言應時之所求合世之所望義存精當妙在長生也)。】
「
所以姑
務契理不取於文。」
【(且務契彼真一之理故不取於藻麗之文)。】

「貽尓方來悟夫懸解者矣。」
【(庶幾貽厥將來好古
博雅之上悟夫逍遙懸解之旨也)。
御製逍遙詠卷第一(五言)】

「　逍遙知語默」

【(聖人之御理也不凝滯於物不封執於教發言合道
守靜惟真所以語默一致逍遙自適也)。】
「　境外見真空」

【(真空之旨超越
塵境故須遊心於杳冥忘情於物象妙用無際綿綿亘古焉)。】
「　有位皆玄感」

【(處非常之位明非常之道皆符契上玄俯順庶品故使樂推長保永福也)。】

「　無緣不用功」

【(妙本冲寂修之仗緣其或真性未明塵心尚壅縱勤
功用事亦虛勞矣)。】
「　義說三才理」

【(天地剖判人倫攸依淳風既微
禮教斯著三才位而萬事彰矣)。】
「　幽深萬事通」

【(極深研幾退藏於密
此乃道體幽微雖通濟萬物而日用不知矣)。】
「　因依堪法則」

【(道經云執
古之道以御今之有蓋行無為之化以理有為之俗􃚳而則之何淳素之不復哉)。】

「　荅謝向旻穹」

【(天之無恩而大恩生故以萬物為芻狗盖不責其報
然覆載之內誰不尊戴於穹旻矣)。】

「　大道虛無境」

【(大道冲粹凝湛一真應用無方本乎妙極虛通之要
盡契玄元矣)。】
「　信之則有靈」

【(上士勤行信而有徵神靈谷盈善得
道用者矣)。】
「　幽深玄妙理」

【(抱一之功幽隱深邃照用玄覽體括虛無
應物自然究理何極)。】
「　運轉幾曾停」

【(天地權與妙本見氣滋養
萬物運轉無窮動而愈出故未暫停也)。】
「　拘束從何益」

【(道之所宗
逍遙任性變通為用虛極為玄或見滯於情事有拘執則有何益矣)。】
「　周通靜杳冥」

【(杳冥之理生成之用周通萬彙動息常靜矣)。】
「　真空非作相」

【(真空之妙非色非形取相而觀背乎道矣)。】
「　善應保安寧」

【(善慶
之功旌乎感應天清地寧自然長久矣)。】

「　我自常安樂」

【(逍遙之道理契希夷達其深源自然安樂矣)。】
「　凡情勿笑耶」

【(道本淵奧宗旨甚深下士聞之而大笑故誡之矣)。】
「　雲生諸洞府」

【(洞府皆神仙棲隱之方故雲霞渺邈非洞府而不生)。】
「　𣗥不放仙花」

【(根性既珠仙凡自異諒其棘刺不秀名花盖明修道之士
品類有殊矣)。】
「　地卑三光照」

【(地本卑下而能厚載得一以寧功契
上玄合三光之照矣)。】
「　天開五色霞」

【(天法於道道法自然運用生成
於物不滯乃五色霞光縱橫分布)。】
「　未知何所是」

【(道本凝寂而昧之者
孰了其元故道經云吾不知其誰之子其斯之謂歟)。】
「　善願理交加」

【(於諸
法中而得至理者乃因善願矣願之與理俱不可廢矣)。】

「　我達逍遙理」

【(觸物無滯名曰逍遙舒卷任情乃達其理隨用
利物其惟聖人乎)。】
「　陰陽運五行」

【(一陰一陽動轉不息潛運五行
之氣用全二儀之功矣)。】
「　道從初一變」

【(混元之前澄寂無狀洎道
生乎一一生乎三變其淳留不出於道也)。】
「　德合混三清」

【(變澆漓之
風歸淳素之化是以大上立德德立道生可以脫三色之樊籠躋三清之聖境)。】
「　聖境堪依仗」

【(抱一還源窮神入聖故境界之超異也復何勝道哉誠堪
依仗耳)。】
「　狐疑事不成」

【(於事不决妄起迷疑如狐之渡冰猶預多
慮志既不一事亦難成)。】
「　異端違雅淡」

【(道性冲淡守之則真其或
攻乎異端是乃違於大道矣)。】
「　返禍自輕生」

【(既不純和貞絜仍聞
順佫背真真境未臻階先及自貽伊感可不謂輕生乎)。】

「　逍遙人控鶴」

【(優游自適至人不拘或駕鶴以凌虛顯逍遙之不滯)。】

「　碧落壓天津」

【(圓盖四垂壓乎四極誠為神仙勝賞之處也)。】
「　達理通含識」

【(物理虛通道性無別一以貫之其在茲矣)。】
「　虛無即是真」

【(虛無妙本體本即真物物周通何往弗濟)。】
「　河車誰肯信」

【(延齡妙術傳乎至人正氣河車信非凡庶)。】
「　鈆汞本相親」

【(鈆汞
相資陰陽共致於道非遠名曰相親)。】
「　造化皆如此」

【(虛而無心動而有應
造化萬彙功成一揆)。】
「　今緣論宿因」

【(積功累德詣道見真諒此今緣
本由曩植)。】

「　何處堪依仗」

【(法天為父稟地為母以德為弓以義為裘用仁作冠
兼信為冕自然堪依堪仗也)。】
「　經心便皺眉」

【(道濟天下功合玄穿
耻不義而取憤不仁而人所以皺眉尓)。】
「　世情多詭詐」

【(狐面狸德
言與行違既乖禮義之風難居仁壽之域故)。】
「　業重信愚癡」

【(易曰
履德之基謙德之柄本乎信崇乎仁豈愚昧而得也)。】
「　取次招他謗」


【(露才揚巳自是他非首招其謗直為怨府尓)。】
「　違非不自知」

【(老氏曰自
伐者無功自矜者不長故行乖德薄自不知尓)。】
「　貌恭無德行」

【(夫教恭柔
侫必外現禮容內荏惡行故宣尼云色厲而內荏也)。】
「　禍福在修持」


【(易曰善不積不足以成名惡不積不足以我身小人以小善為無益而弗為也以小
惡為無傷而弗去也此王在改過修持尓)。】

「　虛無知大道」

【(虛極無為絕於視聽達乎應用其猶橐籥虛而
不屈動而愈出達欺妙者所謂知大道矣)。】
「　極意思悠然」

【(意之極者
所謂忘其象也則神思悠然於物表非世態之所拘爾)。】
「　摩日雲中􃅠」

【(至於道者則情無所滯形無所羈如􃅠在空優游自適)。】
「　瀟湘水似天」

【(瀟湘之水凝碧如天况乎達人性其情而合道忘其象而入無得一混然
良在斯矣)。】
「　孤峯高遠近」

【(峯之高也則洞觀遐邇道之高也則妙入
有無達往知來如視諸掌)。】
「　寶月瑩團圓」

【(皓魄圓時無幽不燭况達
道之士玄覽周通則物無遺照矣)。】
「　信解明開悟」

【(至道冲虛超乎
言象非信解明白者不能開悟於玄奧爾)。】
「　青霄在目前」

【(九重霄漢
渺邈無涯一以貫之宛同目擊庶幾乎不出戶而知天下矣)。】

「　逍遙名有理」

【(夫天命之謂性也率性之謂道也雖道無常名而名
有常理以其綿日月而不老配天地而長存故得謂之逍遙矣)。】
「　得一便知空」

【(既窮真一之門便造真空之境夫如是則得無所得得之大也知無所知知之
至也)。】
「　道本背深邃」

【(源至道之來則持受異氣稟之自然非積
學所能及也信所謂深而且邃矣)。】
「　凡為不易窮」

【(崆峒絕境不可
以苟求汗澷真蹤不可以力致至若心期玉室志慕丹臺者几有所為未易窮也)。】

「　品流才雅淡」

【(飡朝霞而養素吸沆瀣以食精既曰清虛誠惟雅淡
若此之品流即仙才耳)。】
「　挌致意潛通」

【(格致玄同機關密契志以
道寧竟潛通也)。】
「　救世傳相益」

【(万年神衍良足以救世也傳而習之
則有以見復性命之源踐長注之踐斯可謂之益矣)。】
「　因成順八風」


【(吐故納新登神導氣內凝一真之性外調八節之風動順天和咸歸妙本)。】

「　誰測玄機聖」

【(玄機幽邃非凡情之所測惟聖人探賾索隱鉤
深致遠故能顯諸仁藏諸用也)。】
「　天功地久長」

【(範圍天地周備其理
故使能長且久不隨物變尓)。】
「　群生皆是性」

【(群生類聚營務且殊
其間巧拙不侔賢愚莫等論其幻情則有別推其道性則一同)。】
「　道德有圓方」

【(道者同於道德者同於德此乃功用兩全方圓一致也)。】
「　妍醜隨情意」

【(道非美惡情自妍醜老子云天下皆知美之為美斯惡已盖美
善無主俱是妄情矣)。】
「　於戲細酌量」

【(聖人欲令窮理盡性從無入有
故發歎辤誡勸學道之流審細叅詳達於玄奧尓)。】
「　達人知淡泊」


【(寂寥虛靜湛然常存達人體之獨立而不改周行而不殆也)。】
「　默默見非常」

【(非常之道在於無欲而觀是以古先哲王行不言之信處無為之事
玄默之見斯為要妙矣)。】

「　道德誰分別」

「　自是人心所見邪」

「　智識未能通大藥」

【(金丹大藥其旨深玄或智識淺微豈精通深妙也)。】
「　豈將容易擬榮家」

【(一失玄津動皆是妄既無所得何可榮家乎)。】

【(修養之功齊於造化區分擒縱

【(業就功著世人攸仰道光德馨仙籍明錄盖由絕學無為而無
不為也)。】

【(華池發潤金鼎流光伏剛

【(造化

【(人之持論㸦出異端其意頗同是非殊解)。】

「　較量此事盡歸空」

【(較彼是非品量見識未臻大道觸事皆空)。】

「　無為政化求真理」

【(體真理而合無為扣玄門而敷政化盡善

【(不仁而仁仁膈有截

【(夫聖人之動息猶天道之卷舒舉必有常止必有序靜專動直其在茲矣)。】
「　尋思何處不周通」

【(道體周虛道性無礙出處由此無幽不適)。】
「　寰瀛善惡知多少」

【(民之善惡皆由乎心以道治之物無不化)。】
「　莫忘生前積德功」

【(至功至德法地法天民惟樂推史無不載矣)。】

「　因緣業障遇真稀」

【(名未桂於丹臺品未登於仙籙故因
緣尚遠業障尢多所以遇真者稀也)。】

【(史記云黃帝鑄鼎於荊既成有龍垂髯而下黃帝因遂乘之上昇於仙又豈非也)。】

【(有達識者常凝然於妙道故老氏云恍恍

「　愚蒙學解故相違」

【(愚蔽蒙昧之者學寡
洞真解非愽物執以五金八石而為妙者故相違也)。】

【(華池者乃九轉中神水華池也人不虛授天不易求故云秘密也)。】

【(𠃵坤交泰离坎相生契彼三乃應茲九地乃謂

【(天非愛道人亦能修故就無言之言以申無說之說諒非容易務盡玄機庶幾

【(天冠九旒珠光明徹日輪西墜微覺黯然)。】
「　十二樓臺風颺於西波之月」

【(瀛洲山上有十二樓臺皆
神仙之所居也風颺西溟波搖月影下視滄海如掌中焉)。】
「　復還五色雲頭萬𣲖分流」

【(雲含五色散以凝空瑤水分流縱橫萬𣲖也)。】

「　若木之金烏飛乍變」

【(東望飛烏纔升若木壺中瞬息
又見西沉塵西百齡信同隙駒之影也)。】
「　鏘風而白鹿鳴未収」

【(靈毛仙鹿情亦逍遙鳴戲天風呦呦未止)。】
「　眉開八字兮
晶熒煙霞馥郁」

【(眉分八字光彩鑑人鳳腦香濃飄霞拂霧)。】

「　人間非外於縱逸」

【(赤墀高峻丹禁深嚴觸目繁華足以行樂何湏方外始曰勝遊哉)。】
「　終乃是非起而慕求仙」
