* 御製緣識并序　　　　輕

朕聞法門不二。非賢聖無以皆通。
清淨自然。非智慧無以長久。經文
妙覺。識種緣深。誦華嚴而性海分
明。演菩提而真修實行。慈悲喜捨。
法相周圓。窮究化元。絕其虛妄。坦
塗平而無滯礙。去人我而順陰陽。
夫佛理者。思極無涯。遠譯東流之
教。志行方便調御。西域之民莫不
澄寂㳙深。利益真際。恒沙喻於千
界。彼岸混於一法。廣愽談幽。善能
破於昬衢。靈光照耀。豈較量乎深
淺。化導應機。善𢙣隨緣。予將興无
為之大教。力救蒼生之福業。以日
継時。因果必從。常以百姓心而為
念。恢張一切。而行利益。聖人之教
未得美矣。天子之風未得周矣。自
千古以降。迄于今來。寰宇廓清。士
民舒泰。豈敢因循。上天垂莫大之
恩履地。有詠謌之樂恬澹。以虛無
而惟在念。禮讓以明。祀其可交修
泪乎。中衷示信。迺心真而亘南方。
發願興懷。慶雲高而連北斗。隨意
迴向。盡為法界之人。覩境逍遙。變
成華𦙃之國。三輪淨而離塵垢。永
契神清。十善暢而達無生。慈為忍
辱。成功無間。光炟赫於無邊。𨒛遠
益明。觀群生於掌內。燃慧燈於無
極之際。香散雲騰。持寶剎於煩惱
之中。詮深妙用方。知定力真如。猛
利以從心。去住自由。妄想求虛而
逐影。謾作迷情之津要。湛然色相
以安閑。類聚煙蘿。㽵嚴灌頂。歸依
諦信。紀作序云。
御製緣識卷第一

「　虛無知大道極意思悠然摩日雲中􂿎瀟湘水似天孤峯高遠近寶月瑩團圓信解明開悟青霄在目前」


「　往事興人感俳佪思寂寥湛然窮古道群品似相邀春臺松栢靜秋風海岳搖時遷何倏忽落葉自零飄」


「　誰測玄機聖天功地久長群生皆是性道德有圓方妍醜隨情意於戲細酌量達人知澹薄默默見非常」


「　汝看浮生業朝歡暮亦悲笑迎先瞑目𢙣至聚愁眉正直迴斜曲冤逢不用推小人心似獸君子定無欺」


「　恍惚人難曉天仙語默玄修心無道理終日自忙然我命真中趣長生認宿緣機權饒使用聖事不虛傳」


「　大道常恭謹明靈不降威少言知有信多事自招非年壽看松栢冠簪短布衣我心忙也樂清淨得同歸」


「　清靜真空理不無解者難縱撗依見性廓落智中觀坦蕩歸仁德辛勤久遠看勿教為逆順乎地入雲端」


「　寂靜人謌詠愚情豈易知但求心目樂不怕面相欺枉直回邪
滅顛狂道德持高眀常鑒照方寸欲何之」


「　天界殊勝理凡愚不易知曩因誰可究淺近故懷疑的實歸真性區邪入正持樂耶親大道報應自然之」


「　浮生人怕死螻蟻命何輕福業隨緣寂慈悲響善名在家行孝愛食祿盡忠貞天道不移改佛心化有情」


「　逍遙賢聖樂我悶俗情愚鄙俚何堪數斟量解卷舒𢙣緣湏盡去善事好相於忙急真如夢乹坤造化殊」


「　謙遜眀然得滋章道理踈榮枯皆定分巧拙亦何如撫俗從仁感臨民用教敷高明歸輔德建立紀功書」


「　形影隨人得精修道不迷窮深理路造化隱端倪雲引龍吟穴風生虎嘯溪少知愚豈解南比與東西」


「　無窮思廣大幽隱便知深智達千經論方開一寸心聦明懷往古曩刧順從今此理人湏信周旋子細尋」


「　伏藏終不見巧拙是兼非至道光懷古周通隱聖機清天何物性濁地故相依修鍊真堪惜踈慵有順違」


「　為人能省巳就靜亦皆忙春去秋來定年諸日遠長英姿終有異賤品足顛狂蠢動含靈性善生𢙣自亡」


「　知之修鍊法便是聖中人有象虛無境邪緣不見真恒持功行益響應甚平均渾然天地理大道遍乹坤」


「　庶事情多感光陰自古長貪癡何意遣愚軰為誰忙紅葉隨時變白頭㝡易傷世途能及物善德遠馨香」


「　清虛閑靜得奉道見真歸契合皆堪用因緣事好依精誠求諦
信湏有是兼非豁達幽深遠奧妙入玄機」


「　終日窮經史思惟事轉深公忠懷士意是理好相尋聦慧玄高鑒因邪抂用心從容閑嵗月學古便知今」


「　真境堪依仗周遷四敘移人忙閑聖跡心遠罷狐疑俊造皆流品逢能即是師二從寒暑變龍浴鳳皇池」


「　一逢又一別來往幾時休處世皆名利貪生去不留善緣看似夢𢙣事勿臨頭如病心常怕生前樂自由」


「　頂戴尊天地聦明識見高無為通理性猛利豈虛勞出沒終難認因緣不可逃中原多聖境四海湧滔滔」


「　避罪勝修福其為理一般資財豊利益公道不欺瞞濁世貪榮祿明時善政官遏強能撫弱容上勢情觀」


「　如水澹交踈尋思道更殊乖張顛覆易知巳便相於路坦如平掌風傳語不書煙霄明月靜負志話清虛」


「　天上與人間賢愚不蹔閑精修歸至理道大隱玄関聖境皆奇絕凡情豈易攀群仙寬坦蕩却老盡童顏」


「　世途含萬象稽古便知今大樹垂清影如花耀碧岑虛勞空自白德行勝黃金有過而能改無為積善深」


「　法演雙林下凡情世眼澹煙光雨際孤月淨雲端意思皆時暫縱撗大教寬含情能息慮物外不相干」


「　表裏分明說深窮妙更玄愚迷行不到聖事法中傳欲界情無定真空本自然今生隨曩刧清淨照三天」


「　愛老及憐貧逢迎善者親道高明似日世有幾般人盛暑觀
蚋窊魚探海津閑忙皆自得冬去後知春」


「　多言心不善出口莫能追如夢歡娛樂歸家說盛衰擬捋何所用理乱別公私方寸巧施設小人亦可悲」


「　雅澹玄中得常人故不知幽深無遠近道外細推之碧落神仙境煙霞日月隨㕘詳寰宇內邪正豈相離」


「　世間何所益樂善有馨香貴賤雖差等非違定不祥春臺冝勝地花木敗嚴霜求取立身行恒持但久長」


「　逍遙能自在大道亦何殊澹薄隨情遣優遊靜𢙣除春生凡作則秋謝徧悲踈利益玄穹鑒公平信史書」


「　紫陌紅塵裏貪名利好爭世情皆擾攘大道甚縱撗絃直終無曲心邪自不平若能如遂意學法先生」


「　民間天上事通理一言之仙積千年壽人終百嵗期禮容分次第汎愛別尊卑福業有輕重那堪乱作為」


「　世上多虛飾方知一切心不群終是異僻學古同今逸士寬懷耀為官理道深移時非獨見令我思沉吟」


「　真一生天地因依認宿緣智高通理性凡下少周旋語默何奇絕幽深隱聖賢誰人解我意大道寸心玄」


「　石室巢新鷰人心在眼前飛雲高去影敗葉擁流泉取捨憑誰定升沉勿自專守貧知大道盛事好攀緣」


「　豁意何清淨愚人不信迷石巖孤野鶴仙路躡雲梯棟木生深谷寒鴉入夜啼怡顏能待客沽酒旋烹鷄」


「　利鈍猶根性無緣枉用心邪求皆是相正定理玄深巧拙高含
識輕浮力不任大乘通語默稽古便知今」


「　智使千般用思量也大愚黃昬直至曉皎日瑩康衢瑞典通三界華胥樂四隅諭開方便治平坦絕崎嶇」


「　皆救眼前急錐刀蘊利名愚心違善道慧眼自分明大海波浪沒深山棟木成旁求存古意後學法先生」


「　得道超凡聖逍遙本是緣群迷難解會我且不𠁅傳伹信恒無断精窮用意專審𦗟明譬喻勿被俗情牽」


「　古來高逸士􂐴目幾多人積德千花秀溫良七寶珍緣因終可作好事福齊臻宇內光陰速仙天夜月春」


「　君子懷幽趣謙恭禮樂才經心皆識見書史盡通該有德馨還遠清虛道亦開先生冝法則宿習自捋來」


「　遁跡深巖谷身閑未是閑迹窮堪語默寂靜杳真間秘密逢邪見難開大道金丹無効用弗可返童顏」


「　人本心田靜機関買是非聦明存莭操曲直自相依思寂知玄奧區分有順違精通含識性理義愷然希」


「　行止由修鍊輝光朗自然深山藏虎豹大海隱龍泉鶴唳冲霄漢龜生得壽年怡情含道識浩渺九重天」


「　誰人信我說我說亦非常苦行分明樂緣深解即忙乖持如似夢慕道得清涼影響隨聲應玄門可度量」

御製緣識卷第二

「　高山石室半空嵌選取靈芝草盡芟法意要修心一等道情焉用口三緘丹砂保重開清境白髮相冝倚翠巖曩刧緣中因種在布衣鶴神鳳來銜」


「　歸依妙法要精專極目煙霄路不偏日月明通三界眼雲梯上有十洲仙慈悲願力彌何遠煩惱能除
道㝡堅救苦觀音名利益經文留與世間傳」


「　修心慕道入玄機方便門停次苐依種性莫教拘執定因緣去住不相違離他少欲陰功積淡薄深來絕是非浮世利名常自遣真空妄想一時歸」


「　耳根清靜本來修萬法從心不外求海岳動搖山色異玄珠盡在眼中收持齋道士尋紅藥醉臥仙人未白頭麋鹿同群方可信逢春但笑樂花秋」


「　忘機悟即見真詮慕道威儀立善緣語默但教方外用玄微通達自彌堅利名染著還如夢性相歸來猛似泉三業之中堪發問勿令萬法不精專」


「　隨機發問絕親踈法界明文本不殊猒苦群生含萬像歸依大覺達三無蓮開湛寂真如境屋裏空勞捴是虛緣起性中平等用真湏樂道善心居」


「　道高人世禮周旋語默如癡似不言洗滌要教心地靜狐疑勿類水蓱翻經書說盡修真理嵗月休防罷引援雅合無為深見識陰陽去住促寒暄」


「　神丹妙道故延年豈是凡流識大仙老子尚猶能諱事軒皇不許亂相傳誰知造化功夫力罕見今生度宿緣猛虎伏來龍駕馭雲騰五色思悠然」


「　大道逢來是宿緣終期定可訪真仙山中修鍊非州郡城裏無荒隱市䣑秘術豈教容易得幽深還許遞相傳超凡入聖渾閑事問取人間十洞天」


「　青松本不樹蕭條學士因何智自勞金水朗然明皎潔雪山堆積就中高志心覓三神藥凡目難觀萬里毫我獨閑人誰解意悠哉麗日壓靈鼇」


「　天地為爐我自知人無遠慮必生疑功失未石勞懸想境外聲問不
改移機巧豈逢容易得是非終究欲何之思量大海深淵理光陰虛度影參差」


「　崑崙山上聚神仙羽駕爭行意不偏積雪遠看欹下界巔峯高認湧流泉真空的實難堪比異景皆從造化遷秘密豈教容易辯玄中之外更深玄」


「　溟溟滄海浪濤深霧蔽山巒不易尋紫盖洞庭人駕鶴虹霓城郭道為林幽玄相去何多少巧拙無由智慧侵盡是同緣相引接從容語話八仙心」


「　彩雲輕舉𡑅天津舞鶴迎鸞自在身蟾影枝開三島樹桃花風弄十洲春相傳靈異無虛說道理分明盡是真但信此中深護慎修行所願勿因循」


「　天非愛道許人修學士愚迷不到頭內境擬將何所用杳知深邃故難求丹霞隱密堪依仗雪月蹤橫入品流消息湏然潛世慮几故聖
路有剛柔」


「　逍遙裏外混諸天澹蕩其中四敘間拂拭塵埃明印月徧駈神鬼鏁驪山裝添出沒何奇異秘隱藏真未是閑但好消詳清景內勿生疑慮動機關」


「　龍虎剛柔隱聖方一時和合運陰陽周知配類無踈失物性從來理自長鼎裏花開裝五彩心明火候制三黃更能通得華池妙南北東西不可當」


「　水精宮殿月華明白玉為墻火不侵八境風高駢羽駕九重天上絳霞深物情晃朗無遮隔非是凡間比類尋汗澷真遊誰得到桂花香潔影森森」


「　還元到即見真歸不是虛傳羽化飛大勿將閑設用精修別得便忘機安排穩入龍宮內變易終藏虎鼎威豁達𠃵坤神妙理逍遙鈆汞共相依」


「　泥丸本是命根宮消息臨時語話
通大道比來行氣術其餘効驗籍陰功物情之理皆清淨造化玄微事莫同珍重此言深可意腹中日運鼎花空」


「　天生顏㒵幾何般鶴骨飡霞厭俗觀赤水華池誰法則白銀為闕又堪看來傳音旨俱然混立有尊卑勿異端逆順之中開理性心田靜外豈相干」


「　逍遙我命在玄穹鶴宿霞栖景致中自得安寧兼養道更將利益屏群雄淵深引古知今用劈斫區分盡可通和暢五音從豁達乹坤之內靄溟濛」


「　巔峯一望水深潭誰解緣情語話堪暗室自開明日月道心奇異老山菴研窮妙理真消息誡忌先賢讓不貪稽古尚為今法則愚蒙太甚苦談談」


「　非人口似瀉汾河此軰塵中有甚多心恨㒵恭無定止舌誇端正亂乖訛為官善政身榮祿利劒休教
石上磨立待滿盈舩䧟水憑風從被海神拖」
