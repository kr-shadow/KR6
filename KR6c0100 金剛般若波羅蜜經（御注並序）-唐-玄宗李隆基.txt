* 金剛般若波羅蜜經注序(御注并序)

述作者眀聖之能事也。

朕誠寡薄豈冝空為好古竊比徒自為矜飾盖欲弘㢡風教尒昔歲述孝經以為百
行之首故深覃要旨兾闡微言不唯先王至德實謂君子務本近又賛道德㐲知聖祖垂教著經稟訓况道象使人精神專一動合無為凡有以理天
下之二經故不可闕也今之此注則順乎来請夫眾竅互作鼓之者風也柤梨相殊可口者味也苟在將助我者何閒然乎且聖人設教以盡理因言以
成教悟教則言可忘得理而教可遣同乎大通者雖分門而一致攻乎異端者將易性於多方諒在乎不著人我不住福德忘心於三伐閈境於
六塵以音聲求如夢幻法故發菩提者趣於中道習無漏者名為入流將會如如故湏遣遣然三同證皆眾妙門可不羙歟可不羙歟文關事迹
理涉名數注中粗舉而未盡明及經中梵音應湏翻譯者並詳諸義訣云。
「金剛般若波羅蜜經。」
【(可以稱常)。】
「
如是我聞一時佛在舍衛國祇樹給
孤獨園。」
【(如是勝法我聞此一會之時在舍衛國祇陁施樹湏達買園莊嚴道場如是說法)。】
「
與大比丘眾千二百五十人俱。」
【(上果乞士其數若斯隨佛經行得預今會)。】
「
尒時世尊食時著衣持鉢。」
【()。】
「
於其城中次苐𠧒已還至本處飯食訖𠬧衣鉢洗
足巳敷座而坐。」
【(徒而行故洗足將欲說法故敷座)。】
「
時長老湏菩提在大眾中即從座起偏袒右肩右膝著合掌恭敬而。」
【(…解空…明…)。】
「
希有世尊如来善護念諸菩薩善付囑諸菩薩。」
【(護持念恤
成孰根性付託囑授宣通法音功力如斯實為希有)。】
「
世尊善男子善女人發阿耨多羅三藐三菩提心應云何住云何降伏其心。」
【(前歎護之心諮)。】
「
佛言湏菩提如汝所說如來善護念諸菩薩善付囑諸
菩薩汝今諦聽當為汝說善男子善女人發阿耨多羅三藐三菩提心應如是住如是降伏其心。」
【(哉光標)。】
「
唯然世尊願樂欲聞。」
【(受勅諦聽故云唯然深好法音故云願樂)。】
「
佛告湏菩提諸
菩薩摩訶薩應如是降伏其心所有一切眾生之類若𡖉生若胎生若濕生若化生若有色若無色若若無想有想非無想我皆令入無餘涅槃而滅度之。」
【(𡖉胎濕化依四緣生
有色無色依有無生有想無想依境界生若非有想若非無想依麁細觀生如是眾緣皆非中道故我以般若正觀皆令悟入圓寂常樂則滅盡苦趣超度愛流夫以般若智破煩惱障盡般若智存斯有餘也若能遣之又遣空法亦空一切皆如離涅槃相湛常圓寂故曰無餘)。】
「
如是滅量無數無邊眾實無眾生得滅度者。」
【(眾生性本本來寂滅妄心取相開此万緣今但滅妄心令照性本尒豈復別有眾生受滅度耶)。】
「
何以故若菩薩有
我相人相眾生相壽者相即非菩薩。」
【(般若正智無顛倒心計有我人即非菩薩未悟求悟故云令入入則無餘誰云滅度)。】
「
復次湏菩提菩薩應無所住行於布施所謂不住色布施不住聲香味觸法布施。」
【(此以法空荅前住義諸法無相不應取著故菩薩於法無所住心六度撿行六
塵攝境二法空湏三施自忘則不捨之壇於羣有矣)。】
「
湏菩提菩薩應如是布施不住於相。」
【(應如前義離六塵忘二相也)。】
「
何以故若菩薩布施德不思量。」
【(菩提之因無住之施福平等有如虛空故叵量也)。】
「
湏菩提於意云何。」
【(佛言如是之義於汝意謂以為云何)。】
「
東方虛空可思量
不不也世尊湏菩提南西北方四維上下虛空可思量不不也世尊。」
【(無住之施其福甚多借喻虛空將以推挍)。】
「
提菩薩施福德亦復如是不。」
【(福無相是名無住無住之施福如虛空虛空不可量則無住之施亦不可思量也)。】
「
湏菩提但
應如所教住。」
【(所教即上不住相等教也前問應云何住荅云於法住即明菩薩住無所住但依無所住即是如所教住也)。】
「
湏菩提於意云何可以身相見如來。」
【(明住證)。】
「
可以身相得故如來所說身相即非身相。」
【(身相是
生見法身疑寂本無去來眾生見者悉皆虛妄故所說身相皆為眾生妄見既除法身非相)。】
「
佛相非相即見如來。」
【(如來法身無生滅相見有生滅從虛妄生若能悟法性等無生滅凡見諸相皆非真則圓寂法身
離相而見矣)。】
「
湏菩提白佛言世尊頗有眾生得聞如是言說章句生實信不。」
【(凡夫心相生若深聞是處此)。】
「
作。」
【(若為最上乘者說聞說悟不此問也)。】
「
如來滅後後五百歲有持戒修
福者於此章句能生信心以此為實。」
【(後五百歲像法之中具戒福此能信今正法流通而無信耶)。】
「
當知是人不於一佛二佛三千所種諸善根。」
【(能信深法必藉宿因故知是能信之人所種善根非正非一佛二佛也)。】
「
聞是章
句乃至一念生淨信者。」
【(不見信相是名淨信持福能生信心惠解能無見相)。】
「
湏菩提如來悉知悉見是諸眾生得如無量福。」
【(般若智…福德)。】
「
何以是諸眾生無復我相人相眾生相壽者
相無法相亦無非法相。」
【(無我相者以達人空無法相者以達法空無非法者以達空空一念淨信三空解若計諸相則不名淨信能信般若知達法空故知是眾生無諸相也)。】
「
何以故是諸眾生若心取生壽者。」
【(諸眾生應不取者若心起念取相即人計有我人非達般若不能生淨信)。】
「
若取法相即著我
人眾生壽者。」
【(若取諸法計五即於生我相)。】
「
何以故若取非法相即著我人眾生壽者是故不應取非法。」
【(不法…空不有相不…法)。】
「
義如來常說汝等比丘知我說法如筏喻者
法尚應捨何況非法。」
【(以是不取於相故如來常說說無所說豈取相耶故汝等比丘知我所說法本破於有若知有不有悟於空無是名達解以岸以取魚魚筌到岸捨故大聖說空以諸見諸既除空法盡非空法)。】
「
湏菩提於意云三藐三菩提耶如來有所說法耶。」
【(前令捨法遣彼兩偏則佛於菩
提如何獨得而為人說)。】
「
湏菩提言如我解佛所說義無有定法名阿耨多羅三藐三菩提亦無有定法如來可說。」
【(………)。】
「
以故如來所說法皆不可取不可說非法非
非法。」
【(如來所說妙法故不可取離言故不可說而妙契圓證則未嘗不取未嘗非非則不無不有不無諸言相故法)。】
「
所以者何。」
【(所以佛說法而無定法可說者何)。】
「
一切賢聖皆以無為法而有。」
【(根識不同一演說淺深隨分未悟悟藥病皆除無法可說)。】
「
湏菩提於意云何若人滿
三千大千世界七寶以用布施。」
【(將欲挍量持經之功故舉若人財寶之施以明章句理實非虛)。】
「
是人所得福德寧為多不湏菩提言甚多世尊。」
【()。】
「
是福德即非福德性是故如來說福德多。」
【(所以施廣而福多由
非福德性故若是福德性則福由福外緣是故如來說福德)。】
「
若復有人於此經中受持乃至四句偈等為他人說其福彼。」
【(之福非福雖少能)。】
「
何以故湏菩提一切諸佛及諸佛阿耨多羅三
藐三菩提法皆從此經出。」
【(般若正智圓照無礙無上正道因是發明諸佛始則自修終以化物利濟弘何莫由斯)。】
「
湏菩提所謂佛法者即非佛。」
【(自性證則彌同因不滯空直論)。】
「
菩提於意云何湏陁洹能作是念我得湏陁洹果
不。」
【(無證果無名湏陁乘之我般若之忘相)。】
「
湏菩提言不也世尊。」
【(是人不起得果之)。】
「
何以故湏陁洹名為入流而無所入不入色聲陁洹。」
【(無漏果則名入道流不取果相故無流可入豈以忘相而無道可入復於有我而入聲色六塵耶)。】
「
湏菩提於意云
何斯陁含能作是念我得斯陁含果不。」
【(欲界思惟得人我空證果之時有斯念不)。】
「
湏菩提言不也世尊。」
【(是人不果念)。】
「
何以故斯陁含往來是名斯陁含。」
【(盡天人往來生一斯往來明所得果不取果相故實無往來若見一往來果是生我則非斯陁含也)。】

「湏菩提於意云何阿那含能作是念我得阿那含果不。」
【(斷欲界思盡更不復來證果之時有斯念不)。】
「
湏菩提不世尊。」
【()。】
「
名為不来是而實無来是故名阿那含。」
【(達無人我不復來故名不來人法二空
)。】
「
湏菩提於意云何阿羅漢能作是念我得阿羅漢道不。」
【(智惠斷除證果之時有是念)。】
「
。」
【()。】
「
無法名阿羅漢。」
【(以斷煩惱名阿羅漢煩惱空斷無所悟所斷復別阿羅漢耶)。】
「
世尊若阿羅
。」
【()。】
「
昧人中㝡為苐一
是苐一離欲阿羅漢。」
【(勝故云第一如是悉離故亦云第一)。】
「
。」
【()。】
「
湏菩提是
樂阿蘭那行者。」
【(我若起念計有得果違無行世尊不應說我是樂阿蘭那)。】
「
以湏菩提實無所行湏菩提是樂阿蘭那。」
【()。】
「
得不。」
【(將明菩薩亦悟於空
却問我昔為菩薩於然燈佛所授菩提當尓之時見有菩提法可得不)。】
「
世尊如来昔在然燈佛所於法實無所得。」
【(悟法得無生忍勤行般若勝果則菩提深入正觀諸佛由之授記)。】
「
湏菩提於嚴。」
【(可莊嚴不)。】
「
也佛土者即
非莊嚴是名莊嚴。」
【(萬行不常畢竟空寂有修習假名莊嚴國土本空無莊嚴相能了無相即以是莊嚴故是名莊嚴)。】
「
是故湏菩提菩薩摩訶薩應如是生清淨不色生心不應味觸法生心。」
【(是以如是生畢竟空寂心心清淨若是法而生心者非
淨心所證之果非畢竟果故不應如是)。】
「
應無所住而生其心。」
【(知色相空心無所住生清淨心生心之名而無住心之相無住正觀惠照湛然是則以不住法住般若中)。】
「
湏菩提譬如有人身如湏彌山王於意云何是身為不。」
【(假如湏彌山雖有名皆非實體為大身破前取色之心故起大身)。】
「
菩提言大世
尊何以故佛說非身是名大身。」
【(觀乎泰山木石同壇和合眾假名為大本無自性何定大耶所以佛說非身是名大身不應生心以取身相如色聲等不可取也)。】
「
湏菩提如恒河中所有沙數如是沙等恒河於意云何是諸恒河沙寧為。」
【(一河沙亦復如彼如)。】
「
湏
菩提言甚多世尊但諸恒河尚多無數何況其沙。」
【(以沙數河河無數況復於沙實為甚多)。】
「
湏菩提我今實言告汝若善男子善女人以七寶滿尒所恒河沙數三千大千世界以用布施得
福多不湏菩提言甚多世尊。」
【(無性之福施廣福多沙河大千其數無量故知所得福至於甚多)。】
「
佛告湏菩提若善男子善女人於此經中乃至受持四句偈等為他人而此福德勝前福德。」
【(財施雖多無益必菩提以此)。】
「

次湏菩提隨說是經乃至四句偈等當知此處一切世閒天人阿脩羅皆應供養如佛塔廟。」
【(更勝義以勸脩行佛滅度後有塔廟遺像若隨說是經兼持句偈則天人當供養如彼塔廟也)。】
「
何況有人盡能受持讀誦菩提當知
是人成就最上第一希有之法。」
【(般若正智能趣菩提若人受持必生信解即知是人能成就無上菩提故云最上第一希有之法)。】
「
若是經典所在之處則為有佛若尊重弟子。」
【(上成就希有之人勝差降如尊重弟子)。】
「
尒時湏菩提白當何名
此經我等云何奉持。」
【(前來所眀皆無定法法雖無定教必有欲斯問)。】
「
佛告湏菩提經名為金剛般若波羅蜜以是名字汝當奉持。」
【(般若了出中道渡貪流超彼岸勝義汝當循名責實依是奉持)。】
「
所以者說般若波羅
蜜則非般若波羅蜜。」
【(說是妄由執見生病既除真法應捨若執持般若煩惱故佛說般若波羅蜜則無定法名般若也)。】
「
湏菩提於意云何如來有所說法不。」
【(以則)。】
「
湏世尊說。」
【(於言無滯終身言而未嘗言於法無取終身說而未
嘗說知我所說則法則言者不)。】
「
云何三千大千世界所有微塵是為多不湏菩提言甚多世尊。」
【()。】
「
微塵如來說世界
。」
【(無自性積彼微塵以成世界)。】
「
湏菩提於意云何可以三十二相見如。」
【()。】
「
如來說三十二相
。」
【()。】
「
等為他人說其福
甚多。」
【(多以身命布施勝經盡無生)。】
「
世尊佛說甚深經典
我從昔來所得慧眼未曾得聞如是之經。」
【(昔得惠眼於有見空今聞是經於吉解空久得深趣將欲起教以示未)。】
「
是人成就第一希有功德。」
【(相是清淨心得清淨心則能照般若真實之
相得茲實相必趣菩提菩提果成即名苐一希有功德)。】
「
世尊是實相者則是非相是故如來說名實相。」
【()。】
「
經典信解受持不足為難。」
【(親受記久悟空法今聞說信解非難)。】
「
若當來世
後五百歲其有眾生得聞是經信解受持是人則為第一希有。」
【(像法之中去了甚深義無出其)。】
「
無我相人相眾生相壽者相。」
【(何以故此人得名第一希有者為不如是等相故能信解經得為第一)。】
「
所以者
何我相即是非相人相眾生相壽者相即是非相。」
【(於我無相妄計有我能了我相非相即不著人眾生壽者矣)。】
「
何以故離。」
【(何以故此人無我則為希有者能離諸相即是菩提故為希有)。】
「
佛告湏菩提如是如是。」
【(印可其說)。】
「
若復有人得聞是
經不驚不怖不畏當知是人甚為希有。」
【(於無為法而有差別如是賢聖優劣不同中小二乘係執因果福德報應聞是深義必當驚怖能不怖巳是希有況能信解而受持耶)。】
「
說第一波羅蜜非第一波羅蜜是名第一波羅蜜。」
【(諸法莫二故云第一到彼岸者為對此盡彼
何有是知諸法但示假名累盡名故非第一於斯了義能入深門故是名第一波羅蜜)。】

湏菩提若善男子善女人以三千大千世界碎為微
塵於意云何是微塵眾寧為多不甚多世尊。」
【(世界之數以為無碎為微塵故知)。】
「
何以故若是微塵眾有者佛不說是微塵眾。」
【(於塵無自性亦合於相分如是名者皆非實有法)。】
「
所以者何佛說微塵眾是名微塵眾。」
【(前云諸微
塵如來說非微塵是名微塵佛前說以證今說是知微塵無自性無自性故非微塵眾不壞伐法說微塵故是名微塵)。】
「
世尊如來所說三千大千世界則非世界是名世界。」
【(塵性不應取世界之相)。】
「
何以故若世界實有者則是一合相。」
【(不識無性以為實有者則是知一和合相和合眾塵以成世界豈有定世界耶)。】
「
如
來說一合相則非一合相是名一合相。」
【(與世界同俱無性故)。】
「
湏菩提一合相者則是不可說但凡夫之貪著其事。」
【(緣和合本無定相無所可說凡夫計有此相故貪著)。】
「
湏菩提若人言佛說我見人見眾生見壽者
見湏菩提於意云何是人觧我所說義不世尊是人不如來所說義。」
【(佛說我見等將破眾生妄情尒是人便以有說是知觧如來所說義也)。】
「
何以故世尊說我見人見眾生見壽者見即非我見人見眾生見
壽者見是名我見人見眾生見壽者見。」
【(佛說四見以破妄情故所說也而說法破妄必約俗諦故)。】
「
湏菩提發阿褥多羅三藐三菩提心者於一切法應如是知如是見如是信觧不生法相。」
【(前來明中道空義故此云
汝眾生若發菩提心者當依我所說如是知見信觧不應於正觀中道而生法相)。】
「
湏菩提所言法相者如來說即非相是名法相。」
【(菩提法豈有一法而有定相故說法相者令離於相於相無相是名法相)。】
「
湏菩提若有人以滿無量阿僧祇世界七寶持
用布施若有善男子善女人發菩薩心者持於此經乃句偈等受持誦為人演說其福勝彼。」
【(前來量皆以七寶布施不如持經功德今明所以勝彼謂如下文不取於相是福德性故也)。】
「
云何為人演說不取於相如如不
動。」
【(諸法性空假立名字假名之下無相可求故說法之人不應取相夢中占夢豈有相耶如義無相故無去來今說不取相則如彼如矣)。】
「
何以故一切有如露亦如應作如是觀。」
【(無為中道畢竟清淨說不取相則如彼如若一切法中作有為觧如彼夢幻非究竟法作如是觀應不取相)。】
「
佛說是經已長老湏菩提及諸比丘比丘尼
優婆塞優婆夷一切世間天人阿脩羅聞佛所說皆受奉行。」
【(聞甚喜)。】注金剛般若波羅蜜經
